# CareerMitra – AI-Powered Career Development Platform

CareerMitra is a full-stack AI-powered platform designed to help students and job seekers enhance their employability through ATS resume analysis, skill-gap assessment, personalized learning paths, mock interviews, and recruiter matching.

The platform bridges the gap between candidates and recruiters by leveraging AI-driven insights, real-time analytics, and intelligent career guidance.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sakshi0313/CareerMitra.git
cd CareerMitra
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id

VITE_AZURE_OPENAI_KEY=your_azure_openai_key
VITE_AZURE_OPENAI_ENDPOINT=your_azure_openai_endpoint
```

### Run Application

```bash
npm run dev
```
